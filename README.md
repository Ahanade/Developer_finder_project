#  DevConnect - Project Matchmaker

DevConnect is a full-stack web application designed for university students and developers to streamline team formation for side-projects. Users can post their project ideas, specify required skills, track how many team members are needed, and share their contact information.

##  Tech Stack
* **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (Vanilla - Fetch API)
* **Backend:** Python, Flask
* **Database:** SQLite3

##  Features
* **Full CRUD Operations:** Create, Read, Update, and Delete project postings dynamically.
* **Live Dynamic Feed:** View all open projects fetched automatically from the database.
* **Team Seat Counter:** Displays exactly how many open positions are left on a team.
* **Direct Mail Integration:** One-click email communication with project creators.
